# entropy_ui

Графическое приложение для визуализации и проведения экспериментов по теории информации и помехоустойчивому кодированию.

## Описание

`entropy_ui` - это GUI приложение на базе [egui](https://github.com/emilk/egui) и [eframe](https://github.com/emilk/egui/tree/master/crates/eframe), предоставляющее интерактивный интерфейс для:

- Расчетов энтропии источников информации
- Анализа каналов связи с помехами
- Визуализации матриц переходов и совместных вероятностей
- Экспериментов с систематическими помехоустойчивыми кодами
- Экспериментов с кодами Хэмминга

## Установка и запуск

### Из исходников

```bash
# Клонировать репозиторий (с сабмодулями)
git clone --recursive <repository-url>
cd lab1

# Запустить приложение
cargo run --package entropy_ui

# Или собрать релиз
cargo build --release --package entropy_ui
./target/release/entropy_ui
```

### Сборка для Windows

См. [BUILD_WINDOWS.md](../BUILD_WINDOWS.md) в корневом репозитории.

```bash
./build-windows-release.sh
# Результат: target/x86_64-pc-windows-msvc/release/entropy_ui.exe
```

## Использование

### Лабораторные работы 1-3

1. Выберите режим "Лабораторные работы 1-3"
2. Установите параметры:
   - Количество сигналов
   - Количество экспериментов
   - Включите/выключите помехи
   - Включите/выключите длительность символов
3. Нажмите "Запустить эксперименты"
4. Просмотрите результаты: энтропия, условная энтропия, взаимная информация, пропускная способность

### Лабораторные работы 4-6

1. Выберите режим "Лабораторные работы 4-6"
2. Выберите тип кода:
   - **Систематический код** (по умолчанию) - с настраиваемой вероятностью ошибки
   - **Код Хэмминга** - с случайной кратностью ошибки (0, 1 или 2)
3. Установите параметры:
   - `k` - длина сообщения (количество информационных разрядов)
   - Количество экспериментов
   - Для систематического кода: вероятность ошибки
4. Нажмите "Запустить эксперименты"
5. Просмотрите результаты каждого эксперимента:
   - Исходное сообщение
   - Кодовое слово
   - Внесенные ошибки
   - Синдром ошибки
   - Результат коррекции

## Особенности

### Систематический код
- Настраиваемая вероятность ошибки
- Отображение производящей и проверочной матриц
- Компактный вид матриц для больших размеров

### Код Хэмминга
- Случайная кратность ошибки (0, 1 или 2) согласно методическому пособию
- Поддержка модифицированного кода с parity bit для обнаружения двукратных ошибок
- Автоматическое определение кратности ошибки
- Коррекция однократных ошибок

## Зависимости

- `eframe = "0.33.2"` - фреймворк для GUI
- `egui = "0.33.2"` - библиотека интерфейсов
- `entropy_fx` - библиотека функций (локальная зависимость)
- `rand = "0.9.2"` - генерация случайных чисел

## Требования

- Rust 1.91+ (edition 2024)
- Для Windows сборки: `cargo-xwin` и Windows toolchain

## Сборка

### Локальная сборка

```bash
cargo build --release
```

### Кросс-компиляция для Windows

```bash
# Установить cargo-xwin (если еще не установлен)
cargo install cargo-xwin

# Добавить Windows target
rustup target add x86_64-pc-windows-msvc

# Собрать
cargo xwin build --release --package entropy_ui --target x86_64-pc-windows-msvc
```
